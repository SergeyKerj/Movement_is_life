# Movement is life

Лёгкое нативное меню-бар приложение для macOS: следит, сколько ты сидишь за компом
непрерывно, и напоминает встать. Активность определяется по системному idle-таймеру
(клава/тачпад) — без перехвата нажатий и без Accessibility-разрешений.

## Что показывает

Иконка в строке меню:

- `🪑 42m` — сижу непрерывно 42 минуты
- `⚠️ 51m` — перевалил за порог «пора встать»
- `🚶` — отошёл
- `⏸` — пауза

По клику — меню со статистикой и настройками:

```
Сейчас непрерывно: 51 мин
Всего за сегодня: 5 ч 10 мин
──────
Порог отлучки      ▸ 1 / 2 / 3 / 5 / 7 / 10 / 15 мин
Порог «встать»     ▸ 20 / 30 / 40 / 50 / 60 / 90 мин
Промигивать каждые ▸ 3 / 5 / 10 / 15 мин
──────
Сбросить счётчики
Пауза трекинга
Выйти
```

## Логика

- Два состояния: **SITTING** / **AWAY**, переключение по настраиваемому *порогу отлучки*.
- Короткая пауза (< порога) — это чтение/раздумье, сидение продолжается.
- Пауза ≥ порога — «встал»: счётчик непрерывного сидения обнуляется.
- Когда непрерывное сидение превышает *порог «встать»*, иконка мигает «‼️ ВСТАНЬ»
  и повторяет это каждые N минут, пока не встанешь.
- Сон/блокировка экрана трактуется как отлучка.
- «Всего за сегодня» сбрасывается в полночь.

## Сборка и запуск

Требуется Swift 6 (Command Line Tools достаточно), macOS 13+.

```bash
./build.sh
open Movement_is_life.app
```

Приложение живёт только в строке меню (без иконки в Dock).

## Структура

- `Sources/MovementIsLife/Tracker.swift` — машина состояний (чистая логика)
- `Sources/MovementIsLife/AppDelegate.swift` — меню-бар, меню, мигание
- `Sources/MovementIsLife/IdleSource.swift` — чтение системного idle
- `Sources/MovementIsLife/Persistence.swift` — настройки и счётчик в UserDefaults
- `build.sh` — сборка релиза и упаковка в `.app`
